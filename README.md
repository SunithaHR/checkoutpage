## Features

### 1. Product Display
- Prominent product banner with gradient background  
- Star rating display  
- Key features highlighting:
  - Secure Payment  
  - 180-Day Warranty  
  - Single Payment  

### 2. Checkout Form
- **Contact Information**: Email, Full Name, Phone with country code selector  
- **Delivery Address**: Country, Address, Zip Code, City, State  
- **Payment Options**: Credit Card and PayPal  
- **Shipping Insurance**: Optional add-on for $4.95  

### 3. Interactive Elements
- Real-time form validation with error messages  
- Country flag selector for phone codes  
- Payment method toggle (Credit Card / PayPal)  
- Fully responsive mobile-friendly layout  

### 4. Order Summary
- Product details with pricing  
- Subtotal, tax, and shipping breakdown  
- Total amount display  

### 5. Trust Signals
- Security badges (Norton, Secure icons)  
- Guarantee message  
- Social proof notification  
  (“24 people are currently checking out”)

## Form Validation Rules

### Required Fields
- Email (valid format required)  
- Full Name (letters and spaces only)  
- Phone Number (minimum 7 digits)  
- Country  
- Address  
- Zip Code  
- City  
- State  

### Validation Types
- **Email**: `user@domain.com`  
- **Name**: Alphabets and spaces only  
- **Phone**: Minimum 7 digits
